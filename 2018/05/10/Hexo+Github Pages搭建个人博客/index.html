Hexo+Github Pages搭建个人博客

本搭建过程基于windows系统　　
　　目前就读于本科CS专业，最近沉迷技术，偶然间朋友托付，需要使用hexo+github搭建一个个人博客，便在网上查阅了一下相关文章，综合多篇内容终于搭建成功。过程中也遇到了不少的问题和疑惑，在网上找了很多资料，终于一个个解决了。现在基于自己的理解，我把搭建过程和绕过的弯子总结一下，希望能对你的学习有所帮助。

###为什么要用Blog

  • 记录自己的学习过程和感悟，将学到的东西尝试讲解，加深对知识的理解和掌握；
  • 归纳、总结、分类知识点，方便随时回忆查看；
  • 拥有自己搭建的个性化blog，假装自己是个geek；
  • 写写文章，提高自己的文学修养和写作能力；
  • 分享自己的观点，展现自己的价值，得到他人的认可；
  • 记录生活中的美好和烦闷，所有的心情都可以在这里被分享和抒发；

###hexo
Hexo(官方网站)说白了就是一个专门用来搭建博客的静态网页框架，文章必须本地编辑，然后上传。有丰富的插件系统可以选择安装，实现个性化功能和设置。
