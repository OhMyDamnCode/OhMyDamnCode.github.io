　　我的 driver 版本和 Chrome 版本都匹配着，也添加了系统变量，网上查了很多，一直找不到问题在哪里，最后自己调试了半天，终于发现：

chromedriver.exe 文件需要和程序在同一个目录下！！！

　　之前我的程序一直放在桌面，chromedriver.exe 在 C:\Profram File (x86)\Google\Chrome\Aplication ，每次执行是执行都会报错，自从我把他们放在同一个文件夹后，再也不用担心报错啦！
